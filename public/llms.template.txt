## Overview

{{VITE_APP_NAME}} is a platform that converts Large Language Model (LLM) outputs into structured, predictable REST APIs. Define your output schema using JSON Schema, and {{VITE_APP_NAME}} handles the prompt engineering, validation, and API hosting. Works with OpenAI, Anthropic Claude, Google Gemini, and custom LM servers.

## How It Works

1. **Create a Project** - Organize your endpoints
2. **Add an LLM Provider** - Configure your API key for OpenAI, Anthropic, or others
3. **Define an Endpoint** - Set instructions, context, and JSON output schema
4. **Call Your API** - Simple REST calls return structured JSON

## API Format

### Execute Endpoint
```
POST /api/v1/ai/{orgPath}/{projectName}/{endpointName}
Content-Type: application/json

{
  "input_field": "value"
}
```

### Response Format
```json
{
  "success": true,
  "data": {
    "output": { /* Your structured output matching the schema */ },
    "usage": {
      "tokens_input": 42,
      "tokens_output": 15,
      "latency_ms": 523,
      "estimated_cost_cents": 1
    }
  }
}
```
